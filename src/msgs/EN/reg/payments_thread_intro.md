This is the dedicated workspace for matching entry-fee payments and other fees (afterparty, borrowed weapons, etc.).

**How it works:**

1. Upload your bank export here as a text file (`.txt`, `.csv`, or any other text format).
   - Exports from different banks can be uploaded as separate files — each is stored independently.
   - Re-uploading a file with the same name replaces only that file.
2. When all files are uploaded, type **"match"** — I will match the payments to registered fencers.
3. Review the result. If something is off, add a hint (e.g. "line 8 is Novak, not Novakova").
4. Once everything looks good, type **"accept"** or **"ok"** — I will fill in the relevant column in the output sheet.

You can return here at any time to upload additional payment files and re-run the matching.
