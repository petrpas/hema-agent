### Pool Assignment Alchemy

I'll help you put fencers into the best possible pools.

1. First we'll talk about how many pools there will be and how big they should be.
2. Then we'll go over the criteria to optimise for:
   - HEMA rating
   - club membership
   - nationality of fencers
   - any specific constraints
     - for example, we want all fencers competing in two disciplines in the first wave, so the second discipline can run in parallel in the second wave, and so on.
3. Based on the chosen criteria I'll propose an assignment for you to review
   - if you'd like anything improved, just say so
4. Once you're happy with the draw, we'll lock the pools
5. I'll then generate overviews and pool start lists

I'll guide you step by step — just say the word.
